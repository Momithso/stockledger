# StockLedger

StockLedger is a tool for managing and tracking stock inventory, transactions, and balances.

## Features

- Track stock in/out transactions
- Maintain inventory balances
- Generate stock movement reports
- User-friendly interface

## Getting Started

1. **Clone the repository:**
    ```bash
    git clone https://github.com/Momithso/StockLedger.git
    cd StockLedger
    ```

2. **Install dependencies:**
    ```bash
    # Example for Node.js
    pnpm install
    ```

3. **Run the application:**
    ```bash
    pnpm run dev
    ```
